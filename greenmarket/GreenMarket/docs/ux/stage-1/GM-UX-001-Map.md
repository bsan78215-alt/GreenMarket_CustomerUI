# UX-001. Map

**GreenMarket Stage 1 — Screen 1**

## 1. UX-описание

### Назначение

Экран Map предоставляет пользователю пространственное представление продавцов, входящих в каталог GreenMarket.

Экран позволяет быстро оценить расположение продавцов, найти интересующий район и выбрать продавца для дальнейшего просмотра.

Экран не предназначен для просмотра полного ассортимента товаров и не используется для оформления покупки.

### Цель пользователя

Пользователь хочет:

- увидеть продавцов на карте;
- определить, какие продавцы находятся поблизости или в интересующем районе;
- выбрать продавца;
- перейти к подробной информации о продавце.

### Основной сценарий

1. Пользователь открывает приложение.
2. Отображается карта с доступными продавцами.
3. Пользователь просматривает интересующий район.
4. Выбирает продавца.
5. Переходит на экран Seller Card.

### Альтернативные сценарии

**Просмотр каталога** — пользователь переключается на экран Catalog для поиска товаров.

**Просмотр списка продавцов** — пользователь переключается на экран Seller List для просмотра продавцов в виде списка.

**Поиск** — пользователь начинает поиск товара или продавца → переходит на экран Search.

**Повторная загрузка** — при ошибке загрузки пользователь выполняет повторную попытку.

| Параметр | Значение |
|---|---|
| Назначение | Начало поиска товара или продавца |
| Отображаемая информация | поисковый запрос |
| Приоритет | Высокий |
| Появление | Всегда |
| Скрытие | Никогда |
| Фиксированная/прокручиваемая | Фиксированная |

| Состояние ошибки | Не отображается |

## 4. Поведенческая модель

**Область поиска** — появляется при открытии экрана; обновляется при вводе запроса; запускает переход на экран Search.

**Область текущего района** — обновляется при изменении отображаемой территории; скрывается при отсутствии информации.

**Область карты** — появляется после загрузки продавцов; обновляется при изменении отображаемой области; позволяет выбирать продавцов; поддерживает изменение отображаемой территории. Состояния: загрузка; отображение карты; отсутствие продавцов; ошибка; повторная загрузка.

**Область выбранного продавца** — появляется после выбора продавца на карте; обновляется при выборе другого продавца; скрывается при отмене выбора; взаимодействие с областью открывает экран Seller Card.
