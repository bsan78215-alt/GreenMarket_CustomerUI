# Документация проекта GreenMarket Customer UI

В этой директории хранится вся основная документация и спецификации проекта.

## Структура каталога docs/

* **specifications/** — утвержденные технические задания и функциональные спецификации.
* **architecture/** — документы, описывающие архитектуру проекта, платформенные решения и интеграцию.
* **reviews/** — результаты анализа, ревью, проверки и рекомендации.
* **research/** — исследовательские материалы, UX-заметки и аналитика.
